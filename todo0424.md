## C. 알림 / FCM
**건드리는 곳**: `services/messaging`, `features/notifications`, `features/profile/notification_settings`, iOS AppDelegate (APNS)

- [x] **[3]** 앱 켤 때마다 FCM APNS 토큰 미설정 에러
  ```
  Failed to get FCM token: [firebase_messaging/apns-token-not-set]
  APNS token has not been set yet. Please ensure the APNS token is available by calling `getAPNSToken()`.
  ```
  → iOS에서 `getAPNSToken()`을 최대 10초 폴링해 등록 후 `getToken()` 호출. 토큰 발급은 `unawaited`로 비차단 처리하여 부팅 지연 방지
- [x] **[6]** 프로필 탭 알림 설정이 실제 시스템/서버 상태와 불일치 → OS 권한을 단일 진실 원천(SoT)으로 재정의. `_reconcile()`이 (사용자 opt-in) AND (OS 권한)을 effective state로 계산하고, 드리프트(시스템 설정에서 권한 회수) 발생 시 로컬 opt-in 정리 + 토픽 구독 해제까지 함께 처리. 토픽 목록 상수화하여 구독/해제 일관성 확보

## 병합 시 주의 사항

- **A ↔ D/E/F/G**: 모두 ARB를 건드릴 수 있음. A는 "앱 이름 3군데"에만 손대고, 나머지는 각 피처 ARB 키만 추가/수정하면 라인 충돌 없음. 가능하면 **A를 먼저 머지** 후 나머지 워크트리 rebase.
- **D ↔ E**: 10번을 나눠 가지므로, archive 모델에 위치 필드가 비어있는 게 근본 원인이면 **D에서 모델/스키마 먼저 정리 → E가 이어받기**.
- **C**: iOS 네이티브(AppDelegate) 수정 가능성 있음. 다른 워크트리가 iOS 설정을 동시에 건드리지 않도록 주의.
